# COMETの仕様
http://www.ics.teikyo-u.ac.jp/wcasl/spec.html

## アドレスの仕様
- 1語16ビット
- アドレス：0番地から65535番地まで

## アドレス（テープ）の実装
- 1セルに書き込めるのは2進数16文字
- 最低でもテープ長は65535は存在すること

## 数値の仕様
- 数値は16ビットの2進数表記
- 負数は2の補数表記

## 数値の実装
- 数値の仕様に従う
